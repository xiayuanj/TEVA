# Privacy Preserving Federated Learning

**7/10/2021**

## User Guide

### Prerequisite

```bash
sudo apt install libgmp-dev
sudo apt install libmpc-dev 
pip install gmpy2
pip install phe
pip install termcolor
```

In addition, `torch` is the framework for training.

### Plain

Run in CPU:

```bash
python main.py --gpu -1
```
